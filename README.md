# Gatsby Starter

> Gatsby Starter with Typescript and Styled Component for rapid prototyping

| STACK                 |
| --------------------- |
| React                 |
| Gatsby                |
| JavaScript/TypeScript |
| Styled Components     |
| ESLint                |

## Install

```bash
npm i
# or
yarn install
```

## Development

```bash
npm run develop
# or
yarn develop
```

## Build

```bash
npm run build
# or
yarn build
```
